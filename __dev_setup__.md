# Setup flow

## Install tools for develop functions.

```bash
mkdir cloudfunctions
```

```bash
cd cloudfunctions
```

```bash
sudo npm install -g npm@8.4.0
```

```bash
sudo npm install -g firebase-tools
```

## Login Firebase account

```bash
firebase login
```

## Initialize Firebase

Reference [here](https://www.to-r.net/media/note-rss/) ...

```bash
firebase init
```
